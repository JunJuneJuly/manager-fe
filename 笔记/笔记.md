![image-20230719225846959](./asserts/2.png)

根据第二个命令行知道yarn的安装路径，再设置 yarn 的全局安装路径和缓存路径

```
yarn config set global-folder "D:\Application\node\yarn\global"
yarn config set cache-folder "D:\Application\node\yarn\cache"

```
